## 論点1: 事業譲渡時の譲渡価格の算定方法

### 相談内容
AI事業（ILゲーム）を将来的にM&Aで譲渡する場合の価格算定について。

### 主な算定方法

| 方法 | 概要 | 適用場面 |
|------|------|----------|
| **純資産法（コストアプローチ）** | 貸借対照表の純資産額を基準に算定 | 資産が多い事業向け |
| **DCF法（インカムアプローチ）** | 将来キャッシュフローを現在価値に割り引いて算定 | 成長性のある事業向け（AI事業に適する） |
| **類似会社比較法（マーケットアプローチ）** | 類似企業のEBITDA倍率等で算定 | 比較対象がある場合 |
| **年買法（中小企業実務）** | 時価純資産 ＋ 営業利益 × 2〜5年分 | 中小M&Aで多用される |

### 税理士に確認すべきポイント
- AI事業の「のれん（営業権）」の評価方法（ブランド力、ユーザー基盤、AIモデル等の無形資産）
- のれんの税務上の取扱い：**資産調整勘定として5年間で均等償却**（法人税法62条の8）
- のれんは消費税の課税資産に該当 → 譲渡金額が大きいと消費税負担も大きくなる
- 適正な譲渡価格の根拠資料（第三者評価の必要性）

## 論点2: 法人間取引・個人法人間取引の注意点

### 2-1. 移転価格税制

| 項目 | 内容 |
|------|------|
| 対象 | 関連法人間の取引価格が独立企業間価格（ALP）と乖離している場合 |
| リスク | 所得の移転と認定され、追徴課税される |
| 対策 | 独立企業間価格の算定根拠を文書化（ローカルファイル作成） |

### 2-2. 寄附金認定リスク

| 場面 | リスク内容 |
|------|------------|
| 法人→個人への低額譲渡 | 時価との差額が役員賞与or寄附金と認定 |
| 個人→法人への高額譲渡 | 法人側で受贈益課税の可能性 |
| 法人間の無対価取引 | 全額が寄附金として損金不算入 |
| 業務委託費の不相当 | 実態が伴わない場合、寄附金認定 |

### 税理士に確認すべきポイント
- 既存法人（VONDS）と新規法人（AI事業用）間の取引ルール策定
- 個人（小沢）と法人間の取引における適正価格の根拠
- 関連会社間の管理料・ロイヤリティの妥当な水準
- 移転価格文書化（ローカルファイル）の要否と範囲

## 論点3: 報酬設計の最適構成

### 選択肢の比較

| 報酬形態 | 税率・負担 | メリット | デメリット |
|----------|-----------|----------|------------|
| **役員報酬（定期同額給与）** | 所得税 5〜45% ＋ 住民税10% ＋ 社会保険料 | 法人側で損金算入 | 期中変更不可、社会保険料が高い |
| **業務委託報酬** | 事業所得として課税 | 柔軟な設定が可能、社保負担なし | 実態が伴わないと否認リスク |
| **配当** | 所得税 20.315%（上場）/ 総合課税（非上場） | 社会保険料の対象外 | 法人側で損金不算入 |
| **退職金** | 退職所得控除あり、1/2課税 | 税制上最も有利 | 退職時のみ |

### 税理士に確認すべきポイント
- 複数法人での役員報酬の配分と社会保険の適用関係
- 業務委託契約の実態要件（指揮命令関係の有無、独立性）
- 代表者への適正な報酬水準の根拠
- 将来の退職金支給に向けた準備（役員退職慰労金規程の整備）

## 論点4: 資金移動の方法と税務リスク

### 法人→個人の資金移動

| 方法 | 税務上の取扱い |
|------|---------------|
| 役員報酬 | 損金算入可（定期同額給与の要件あり） |
| 役員貸付金 | 認定利息の問題（利率：短期貸付金利 年0.9%程度 ※2025年） |
| 配当 | 法人側で損金不算入、個人側で配当控除あり |
| 退職金 | 不相当に高額でない限り損金算入 |

### 法人→法人の資金移動

| 方法 | 税務上の取扱い |
|------|---------------|
| 業務委託料 | 実態があれば損金算入可 |
| 経営管理料 | 合理的な算定根拠が必要 |
| 貸付金 | 適正利率での貸付が必要（無利息は寄附金認定リスク） |
| 配当（子会社→親会社） | 益金不算入制度あり（持株比率による） |

### 税理士に確認すべきポイント
- デリヘル事業の利益をAI事業の投資資金に充てる最適なルート
- 役員貸付金が既に存在する場合の解消方法
- グループ間の資金プール（CMS）の是非

## 論点5: 合同会社 vs 株式会社の選択基準

### 比較表

| 項目 | 合同会社（LLC） | 株式会社 |
|------|----------------|----------|
| 設立費用 | 約6〜10万円 | 約20〜25万円 |
| 登録免許税 | 6万円（最低額） | 15万円（最低額） |
| 定款認証 | 不要 | 必要（約3〜5万円） |
| 決算公告 | 不要 | 必要 |
| 利益配分 | 出資比率に関係なく自由 | 出資比率に応じて |
| 意思決定 | 社員の過半数 | 株主総会＋取締役会 |
| 社会的信用 | やや低い | 高い |
| M&A（株式譲渡） | 不可（持分譲渡は可能） | 可能 |
| 株式上場 | 不可 | 可能 |

### 税理士に確認すべきポイント
- AI事業法人として合同会社を選んだ場合、将来のM&Aスキームへの影響
- 合同会社の持分譲渡と株式会社の株式譲渡の税務上の違い
- 合同会社→株式会社への組織変更コスト（登録免許税6万円＋手続き費用）
- M&Aを前提とするなら最初から株式会社にすべきか

## 論点6: M&A時の株式譲渡 vs 事業譲渡の税務上の違い

### 比較表

| 項目 | 株式譲渡 | 事業譲渡 |
|------|----------|----------|
| **売り手の課税** | 個人：譲渡所得 約20.315% | 法人：法人税 実効税率約30〜34% |
| **買い手のメリット** | 手続きが簡単 | のれん償却（5年）で節税可能 |
| **消費税** | 非課税 | 課税資産に対して消費税10%が発生 |
| **簿外債務リスク** | 引き継ぐ | 引き継がない（選択的） |
| **許認可** | 原則引き継がれる | 再取得が必要な場合あり |
| **従業員** | 自動的に引き継ぎ | 個別に同意が必要 |

### 税理士に確認すべきポイント
- AI事業の譲渡において、どちらが税引後手取りを最大化できるか
- デリヘル事業を残してAI事業のみ譲渡する場合のスキーム設計
- 事業譲渡時の消費税負担の見積もり
- 買い手から見たスキームの選好（のれん償却のメリット）

## 論点7: インボイス制度・消費税の影響

### 2026年10月以降の変更点（令和8年度税制改正大綱）

| 期間 | 免税事業者からの仕入税額控除 |
|------|---------------------------|
| 〜2026年9月 | 80%控除可能 |
| 2026年10月〜2029年9月 | **70%控除可能**（当初50%予定が緩和） |
| 2029年10月〜2032年9月 | 50%控除可能 |
| 2032年10月〜 | 控除不可（経過措置終了） |

### 2割特例の改正
- 2026年9月末で「2割特例」終了
- 2026年10月〜：個人事業者向けに「3割特例」（2年間限定）が新設
- **法人には3割特例は適用されない**

### 税理士に確認すべきポイント
- 新設法人のインボイス登録のタイミング
- デリヘル事業（非課税売上が多い）の消費税計算への影響
- AI事業における簡易課税 vs 本則課税の有利不利判定
- 事業譲渡時の消費税負担シミュレーション
- 法人間取引における適格請求書の発行体制

## 論点8: AI事業にかかる費用の経費計上

### ソフトウェア・AI開発費の税務上の取扱い

| 費用の種類 | 税務上の処理 |
|-----------|-------------|
| 研究段階の費用 | 発生時に損金算入可能 |
| 自社利用ソフトウェア開発費 | 資産計上 → **耐用年数5年**で定額法償却 |
| 市場販売目的ソフトウェア | 資産計上 → 耐用年数3年で償却 |
| クラウドサービス利用料 | 発生時に損金算入可能 |
| AI学習用データ購入費 | 内容により判断（データベース：5年償却） |
| 外注開発費 | ソフトウェアの取得原価に含める |
| サーバー・GPU等のハードウェア | 減価償却資産（サーバー：5年） |

### 研究開発税制（R&D税制）

| 項目 | 内容 |
|------|------|
| 対象 | 試験研究費（人件費、原材料費、委託費等） |
| 控除率 | 法人税額の1〜14%（増減試験研究費割合による） |
| 控除上限 | 法人税額の25%（上乗せあり） |
| 注意点 | **自社利用ソフトウェアの制作費は原則対象外** |
| AI関連 | AI・IoT分野はオープンイノベーション型で上乗せ控除の可能性あり |

### 税理士に確認すべきポイント
- ILゲームの開発費用が研究開発費に該当するか否かの判断基準
- 自社利用ソフトウェアの範囲（SaaS提供の場合の取扱い）
- AI開発における外注費・クラウド利用料の資産計上 vs 費用処理の判断
- オープンイノベーション型の適用可否（大学・研究機関との共同研究）
- 中小企業技術基盤強化税制の適用可否
