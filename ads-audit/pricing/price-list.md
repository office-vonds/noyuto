# VONDS Google広告サービス価格表

**最終更新**: 2026-04-10
**適用開始**: 2026-04-11〜
**通貨**: 日本円(税別)

---

## 基本方針

1. **着手金+月額**の2段構え。着手金で初月キャッシュフローを確保、月額で継続収益化
2. **業界テンプレでパッケージ化**: 料金は業界カテゴリで固定(工数ブレを吸収)
3. **VONDSの22年実績 × Claude Code自動化**がセット:属人性を下げつつ品質は下げない
4. **山梨県内企業には対面サポート込み**(移動費別途)

## プラン1: スポット監査(入口商品・最速現金化)

### 標準プラン ¥150,000(税別)

**想定クライアント**: 月広告費30万円以上・運用歴6ヶ月以上
**納期**: 7営業日以内
**納品物**:
- Google広告診断レポートPDF(20〜30ページ)
- ヘルススコア(0-100)+グレードA-F
- Critical/High/Medium/Low の優先度付き指摘事項リスト
- クイックウィン一覧(15分で直せる改善)
- 無駄遣い検出レポート(具体的な金額)
- 業界ベンチマーク比較
- VONDSからの改善提案書

**作業内容**:
1. MCC接続 or CSVエクスポート受領
2. 74チェック項目の評価実行(Claude Code claude-ads スキル使用)
3. 業界テンプレ(local-service/healthcare 等)との照合
4. 日本語化+VONDSブランド化
5. 代表・NOYUTOの所見を追加(22年の経験値)
6. PDF納品+オンラインミーティング(60分)での説明

### プレミアム監査 ¥220,000(税別)

**追加内容**:
- Meta広告・LinkedIn広告等、他プラットフォームも同時監査
- クイックウィン項目は監査と同時にVONDSが無償で実装
- LP診断も同梱(Core Web Vitals・CVR改善提案)
- 業界競合3社のGoogle広告分析

**想定クライアント**: 月広告費100万円以上・複数プラットフォーム運用・EC業態

### ライトプラン ¥75,000(税別)

**対象**: 運用開始前〜3ヶ月未満のローンチ直後アカウント
**納品物**: ベースライン診断+セットアップチェックリスト+最初の3ヶ月のアクションプラン
**例**: 本気ストレッチ/ストレッチゼロ(設定段階)向け

## プラン2: 改善プラン実行(中間商品)

### 改善パック ¥150,000〜¥300,000(税別)

**内容**: スポット監査で指摘した Critical/High 項目を VONDS が実装
**料金決定方法**: 対応項目数・作業工数による見積もり
- 基本料金 ¥150,000 + オプション項目追加
- 10項目まで ¥150,000
- 20項目まで ¥220,000
- 無制限(全Critical+High対応) ¥300,000

**期間**: 契約後5〜10営業日
**含まれる内容**:
- コンバージョン計測の再整備(Enhanced Conversions/Consent Mode v2)
- 除外キーワードリストの作成・適用
- アカウント構造の最適化
- RSA広告の改善(ヘッドライン・説明文)
- 拡張機能(サイトリンク・コールアウト等)の追加
- ランディングページの改善提案書

## 支払い条件

1. **着手金**: 契約締結時に50%、納品時に残り50%(スポット監査の場合)
2. **月額運用代行**: 毎月末締め・翌月末払い
3. **請求方法**: 銀行振込(PayPay銀行 or 甲府信用金庫)
4. **インボイス**: 適格請求書発行事業者登録済(2024/6/15〜)
5. **キャンセル**: 契約後48時間以内はキャンセル料無し・以降は進捗に応じて請求
