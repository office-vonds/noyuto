# VONDS Google広告監査サービス

## 事業概要

Claude Code + `claude-ads` スキル(74チェック項目・6並列サブエージェント監査)を運用基盤にした、地域中小企業向けGoogle広告監査の商品化サービス。

VONDSの22年のGoogle広告運用経験 × Claude Code のオペレーション自動化により、**1件あたり2〜3時間の作業工数で¥75,000〜¥220,000のスポット売上**を作る設計。

## なぜこれが最速で現金化できるのか

1. **既存クライアント基盤が武器**: SANKEN(買取コンシェルジュ・本気ストレッチ・ストレッチゼロ)、A-TRUCK——すでにMCC配下にある既存クライアントへの追加提案が初回販売先
2. **作業の大半を Claude Code が肩代わり**: 74項目のチェック・採点・Quick Wins抽出を自動実行、人間は最終レビューと日本語化・PDF化のみ
3. **再現性が高い**: 一度テンプレを作れば同じフローで何社でも回せる
4. **信頼の証拠になる**: 監査レポート自体が「次の運用代行契約」への入口

## サービス構造


| 1. 入口 | Google広告スポット監査 | ¥75,000〜¥220,000 | 80%+ |

## Claude Code での実行フロー

### 1. 監査実行
```
/ads google
```
または手動で:
- `audit-google` エージェントに該当クライアントのGoogle広告データを渡す
- 74チェックを走らせ、カテゴリ別スコア+Quick Wins を出力させる

### 2. 日本語化+VONDSブランド化
- `audit-report-template.md` に結果を流し込む
- 表現を日本の中小企業向けに翻訳
- NOYUTO の所見を Executive Summary に追加

### 3. PDF化
- Pandoc または類似ツールで PDF 出力
- VONDSロゴ・表紙・裏表紙を差し込み

### 4. 提案書同梱
- `proposal-existing-client.md` または `proposal-new-prospect.md` を組み合わせ
- 見積書も同梱(`pricing/price-list.md` から転記)

## 成果物の品質基準

- **必ず日本語**: 元スキルは英語だが、納品物は100%日本語
- **必ず具体数値**: 「改善余地あり」ではなく「月額¥XXの無駄遣いを特定」
- **必ず優先順位**: Critical→High→Medium→Lowの順番で提示
- **必ずQuick Wins**: 15分以内で実装できる改善を最低3つ提示
- **必ず次のアクション**: 最後に「VONDSが実行する場合」の見積もりを提示

## パパの運用ルール

1. **見てから語れ**: 監査前に必ずクライアントのアカウントを自分の目で確認する。Claude Code の出力を鵜呑みにしない
2. **推測で数字を書くな**: 「おそらく月¥10万の無駄」ではなく、GAQLで実データを取ってから書く
3. **既存クライアントには正直に**: 「Claude Code を使って自動化してます」は隠さない。むしろ武器として前面に出す
4. **レポートは必ず git にコミット**: サンプルが資産になる
5. **バックアップを取れ**: 本番実行前に git commit
