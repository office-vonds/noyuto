## 🚀 最重要発見: ストレッチゼロの広告最終URLを変えるだけでCPA半減可能

### Paid Search ページ別パフォーマンス（30日）

| 流入ページ | Sessions | CV | CVR | エンゲージメント |
|---|---:|---:|---:|---:|
| **`/shop/`（店舗一覧）** | **182** | **70** | **38.5%** | **92%** |
| `/`（TOP） | 830 | 0 | **0%** | 48% |
| `/trainer/` | 35 | 0 | 0% | 97% |
| `/contact/` | 26 | 0 | 0% | 100% |
| `/renewal/` | 19 | 0 | 0% | 100% |
| `/shop/shop-kofu/` | 18 | 0 | 0% | 94% |

**CV対象イベント**: `reserve_click` 209件（122ユーザー）

### 判明した事実

- **Paid流入でCV出てるのは `/shop/` のみ**（Paid全CV70件=/shop/ 70件、TOP流入CV ゼロ）
- **TOP（/）は Paid 830セッションあるが CVR 0%**（エンゲージメント48%・完全に着地したが予約せず離脱）
- **/shop/ はCVR 38.5%・エンゲージメント92%**（店舗一覧 → 予約クリックの導線が機能）
- 現状: 広告の多くが TOP に着地して無駄撃ちになっている

### 改善案（即実行）

**Ads Editor で全KWの最終ページURLを `https://stretchzero.jp/shop/` に変更**

試算:
- 現状: 30日 ¥117,681 / CV 70 → **CPA ¥1,681**
- 改善後推定（CVR 38.5%が主力なら）: 同予算でCV推定 150-200件 → **CPA ¥600-¥800**
- **CPA半減 = 同予算で CV倍増 or CPA維持なら月¥50,000削減可能**

ただし実運用では CVR がTOP着地でも平均化するので、**仮定を保守的にしても CPA 20-40%改善**は固い。

### 他にやるべき優先度中タスク

- `/trainer/` (35sess CV0) → トレーナー紹介ページで予約導線が弱い → CTA追加 or `/shop/` リンク強化
- TOP → `/shop/` への誘導CTA強化（TOP残しつつエンゲージメント→予約に変換）

## 🔴 本気ストレッチは広告自体がほぼ機能していない

### 過去30日 全体

- Users: **120** / Sessions: **195** / PV: 388
- **Paid Search セッション: たった 7件**（同期間のストレッチゼロは744件・**100倍差**）
- **GA4 CV対象イベント: 定義ゼロ件**（conversion mark未設定）

### Paid Search 詳細

| 流入ページ | Sessions | CV | エンゲージメント |
|---|---:|---:|---:|
| `/` | 7 | 0 | 14% |

**7セッションしか届いていない** = 予算5,000円/日に引き上げても、そもそもKWがヒットしていない・入札で勝てていない・広告文で選ばれていない。

### 原因の仮説（要検証）

1. **KW数が少ない**（ストレッチゼロのようにTier1-2のKW11本を追加する必要あり・棲み分け考慮）
2. **入札単価が低すぎて競合に負けている**（Paid Search 7セッションの時点でほぼ誰も見ていない）
3. **広告文の質が低く、CTRが極端に低い**
4. **GA4のCV属性が未定義のため、スマート入札が機能していない**

### 即実行すべき改善

**段階1（即日・俺で実行可）**
- 本気ストレッチ用KWを棲み分け方針に沿って設計（ストレッチゼロ側と被らない領域）
- キーワード候補: 「姿勢改善」「精密姿勢診断」「可動域改善」「返金保証 ストレッチ」「甲府上阿原 ストレッチ」「マッサージ 卒業」系

**段階2（NOYUTO・5分）**
- GA4 Admin → イベント → `tel_click`, `form_submit_complete`, `cta_click` を「コンバージョンイベント」としてマーク
- マークするとGoogle広告のスマート入札が正しく学習するようになる

**段階3（Google広告管理画面）**
- Google広告管理画面で「コンバージョン列」に本日整理した4CV（TEL/フォーム送信/CTA/メール）を入れる
- スマート入札を「目標コンバージョン数最大化」or「tCPA」に切替え
