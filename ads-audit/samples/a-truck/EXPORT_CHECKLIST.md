# A-TRUCK Google広告 エクスポート手順書(NOYUTO向け)

**対象アカウント**: A-TRUCK株式会社 Google広告
**保存先**: `/home/noyuto/projects/vonds/ads-audit/samples/a-truck/data/`
**所要時間**: 10〜15分

---

## 事前準備

1. Google広告管理画面(ads.google.com)にログイン
2. 左上のアカウント選択から **A-TRUCK** を選択
3. 右上の期間を **過去30日間** に設定

## エクスポートする7つのレポート

それぞれ画面右上の「**ダウンロード**」ボタン(↓アイコン)から **.csv 形式** でダウンロードし、ファイル名を下記の通りにリネームしてから `data/` フォルダに保存。

### 1. アカウント全体サマリー
- 左メニュー: **概要**
- 表示: グラフ下のサマリー表
- ダウンロード → 形式: CSV
- 保存ファイル名: `01_account_summary.csv`

### 2. キャンペーン一覧
- 左メニュー: **キャンペーン**
- 列の表示設定で「**入札戦略タイプ**」「**ステータス**」「**予算**」「**費用**」「**コンバージョン**」「**コンバージョン率**」「**費用 / コンバージョン**」を含める
- ダウンロード → CSV
- 保存ファイル名: `02_campaigns.csv`

### 3. 広告グループ一覧
- 左メニュー: **広告グループ**
- 列に「**品質スコア**」「**広告の強度**」を追加
- ダウンロード → CSV
- 保存ファイル名: `03_ad_groups.csv`

### 4. キーワード一覧(最重要)
- 左メニュー: **キーワード** → **検索キーワード**
- 列に「**品質スコア**」「**推定クリック率**」「**広告の関連性**」「**ランディング ページの利便性**」を追加
- ダウンロード → CSV
- 保存ファイル名: `04_keywords.csv`

### 5. 検索語句レポート(無駄遣い検出用・最重要)
- 左メニュー: **キーワード** → **検索語句**
- 期間: 過去30日
- 費用の高い順に並び替え
- ダウンロード → CSV
- 保存ファイル名: `05_search_terms.csv`

### 6. 広告一覧
- 左メニュー: **広告**
- 列に「**広告の強度**」「**見出し数**」「**説明文数**」を追加
- ダウンロード → CSV
- 保存ファイル名: `06_ads.csv`

### 7. コンバージョン設定
- 右上の **ツールと設定** → **測定** → **コンバージョン**
- コンバージョンアクション一覧を表示
- ダウンロード → CSV
- 保存ファイル名: `07_conversions.csv`

## オプション(時間があれば)

### 8. 除外キーワードリスト
- **ツールと設定** → **共有ライブラリ** → **除外キーワードリスト**
- スクショでもOK(ファイル名: `08_negative_keywords.png`)

### 9. 拡張機能(アセット)
- 左メニュー: **アセット**
- スクショまたはCSVで保存(ファイル名: `09_assets.csv` または `.png`)

### 10. アカウントスクリーンショット
- 概要ページ全体のスクショ1枚(ファイル名: `10_overview_screenshot.png`)
