# NOYUTO 統合戦略書（2026年3月18日時点）

---

## ■ プロジェクトの本質

「発信」ではなく「思想の設計と収益化」。
思想 → 共感 → 信用 → 購買の導線設計。
単なるコンテンツビジネスではなく、"思想を資産化して現金化するモデル"。

---

## ■ 媒体構造

- **X** = 集客装置（拡散・思想提示・フック）
- **note** = 収益装置（教育・思想深化・商品化）
- **引用リポスト** = 最速のリーチ拡大装置（NOYUTOの存在を知らない層に届く）

---

## ■ 設計原則

**「思想は属人、運用は非属人」**

- 思想・哲学・トーン → 小沢宗弘のもの（100%属人）
- 投稿生成・スケジュール・分析 → システムが自動化（非属人）
- 引用リポストの選択と執筆 → 小沢さんの判断（属人）

## ■ X投稿ルール

### フォーマット
- スレッド4本構成
- 各投稿は十分な長さ（短文禁止）

### 構造
1. **フック** → 毒/インパクトで切り込む。スクロールを止める
2. **煽り** → 反論を先回りして煽る。敵と味方を明確化
3. **根拠** → 体験・数字・事実で黙らせる
4. **着地** → 答えは出さない。問いかけで着地。noteへの自然な導線

### 絶対ルール
- **必ず敬語（です・ます調）**
- ハッシュタグ：思想の旗印として1〜2個だけ（例：#信頼経済）。大量付けは禁止
- 絵文字：不使用
- 偏らず断定せず、結論は出さない。ただし刺激とインパクトは最大化
- テーマ（タイトル・フック）は断定する。着地は断定しない
- 5〜10年後も色褪せない普遍的テーマを重視

### テーマの断定度
| パート | 断定度 |
|--------|--------|
| テーマ/タイトル | 断定する（強く） |
| 1本目（フック） | 断定する |
| 2本目（煽り） | 挑発する |
| 3本目（根拠） | 体験で証明する |
| 4本目（着地） | 断定しない。問いで終わる |

### 写真
- 汎用素材写真は不要。むしろ邪魔
- 使うのはNOYUTOが現場で撮った写真、実際の数字、実証データのスクリーンショットだけ

## ■ 引用リポスト戦略

### 位置づけ
- オリジナル投稿はフォロワーにしか届かない
- 引用リポストは相手のフォロワー全員に届く → 最速のリーチ拡大手段
- 自動化しない。NOYUTOの判断が入るべき場所

### 3つのタイプ
1. **攻撃型**：常識・構造を切る（人ではなく仕組みを切る）→ 最も拡散する
2. **思考起動型**：問いだけ置く → フォロワー転換率が最も高い
3. **共鳴型**：本質を肯定して体験を乗せる → 信頼残高を積む

### 覚悟
- **命に関わるテーマは、躊躇なく断定する。遠慮しない**
- ビジネス・社会論では構造批判で丁寧に切る
- 健康・命・生き方では力強く言い切る
- 毒にも薬にもならない思考はクソ。嫌われてでも言い切る

### 頻度
- 日1〜2本（手動）
- 多すぎると他人依存アカウントに見える。少なすぎると機会損失

### 候補の収集
- **システムが自動で収集** → search_quotes.pyが毎朝9:05にX APIで検索 → Gmailに候補が届く
- **小沢さんは選ぶことと書くことだけ**

### 監視キーワード
| カテゴリ | キーワード |
|---------|----------|
| 攻撃型 | 「売上が全て」「結果を出せ」「効率化」「正解は」 |
| 共鳴型 | 「信頼」「行動」「矛盾」「些事」「志」 |
| 時事フック | 「AI 仕事」「健康 正解」「副業」 |

## ■ 参考アカウント学習戦略

### 原則
- 取り入れるのは「他人の思想」ではなく「他人の構造」
- 思想はコピーしない。器の設計だけを学ぶ

### 学習する3つの層
1. **エンゲージメント構造**：ブックマークされる型、リスト形式、常識への反論
2. **フック設計**：最初の一文で止める技術
3. **導線設計**：プロフ→固定ツイート→購入の設計

### 運用
- Notionに「参考投稿ストック」ページを作成（明日）
- 刺さった投稿のURL + なぜ刺さったか + 構造の特徴をメモ
- 義務化しない。刺さった瞬間にストックする
- 5〜10件溜まったらClaude Codeで構造分析 → 生成ルールに反映
- 将来的にはX APIで参考アカウントの投稿を自動取得→分析を自動化

### 参考アカウント例
- @CS60osaka1（健康系、10万フォロワー、ブックマーク率が高い）

## ■ note戦略

### メンバーシップ
- 「第3の視点・思考実験室」¥980/月
- 「答えの出し方」を盗みたい人だけ来る場所

### 記事構造
- 冒頭：Xスレッドとの連動、引き込み
- 本文：体験ベースで深掘り、無料では語らない深度
- 結語：思考実験室への自然な導線

### X→note導線
- 「続きはnoteで」は禁止
- 「この問いの答えを書いた」型のリフレームフック
- CTAではなく「知りたい欲求」を起動させる
