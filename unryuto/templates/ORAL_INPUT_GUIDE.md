# NOYUTO 口述受け入れガイド

## 口述の方法（どれでもOK）

1. **テキスト直接入力** — Claude Codeに話しかけるようにそのまま喋る
2. **音声メモ** — スマホの音声入力でテキスト化してから貼り付け
3. **箇条書きメモ** — キーワードだけでもOK。構造化はサムがやる

## 口述テーマ（求人AI戦略マニュアル用）

以下のテーマを1つずつ話してください。順番は自由。

### テーマ1: バニラ自動投稿の仕組み
- どうやって思いついたか
- 技術的に何をやっているか（ざっくりでOK）
- 設定方法（何をどう設定すれば動くか）
- 結果: 何が変わったか

### テーマ2: ガールズヘブン自動投稿
- バニラとの違い
- 2件の問い合わせが来た時の状況
- 何をしたから来たのか（NOYUTOの分析）

### テーマ3: 地域別の求人SEO戦略
- 山梨でどうやっているか
- 他の地域でやるならどうするか
- 検索されるキーワードの考え方

### テーマ4: 従来の求人方法との比較
- AIを使う前はどうしていたか
- コスト・効果・手間の違い
- 業界の他の店はどうしているか

### テーマ5: AIの使い方（風俗経営での実践）
- ChatGPT / Claude をどう使っているか
- 求人文をAIで書くコツ
- AIに任せていいこと・ダメなこと

## 口述テーマ（営業戦略マニュアル用）

### テーマ6: ILゲームの設計思想
- なぜILを上げると売上が上がるのか
- ILゲームを作ろうと思った経緯
- キャストの反応

### テーマ7: 300PV方程式
- 1成約=300PVの根拠
- 実戦での使い方
- PVを上げる具体的な施策

### テーマ8: 写メ日記戦略
- デリヘルタウンでの写メ日記の役割
- 何枚・どのタイミング・どんな内容が効くか
- 自動化している部分

### テーマ9: キャスト教育
- 何をどの順番で教えるか
- 新人が辞めるパターンと対策
- 離職率26%の意味

### テーマ10: 経営判断のフレームワーク
- 22年で学んだ判断基準
- 数字の見方（何を見て何を決めるか）
- やってはいけないこと

## サムの処理フロー

口述を受け取ったら:
1. テーマ別に分類
2. manual-template.html の構造に当てはめる
3. 章立て → 小見出し → 本文 → ポイント → 実績データ の順で構造化
4. HTML生成 → PDF変換（convert-manual.sh）
5. NOYUTOに最終確認依頼
