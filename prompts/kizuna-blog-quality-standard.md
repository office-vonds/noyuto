## 絆求人ブログ 記事品質基準書（100点基準）

### 目的
「山梨 風俗 求人」で検索する女性が、絆で働くことを本気で検討するための判断材料を提供する。
10年後に読んでも色褪せない、普遍的な価値のあるコンテンツ。
「店が言いたいこと」ではなく「読者が知りたいこと」を書く。

---

### 5つの鉄則（全国リサーチに基づく）

1. **具体的な数字で語れ、抽象論で逃げるな** → 1記事に最低3つの金額・日数・時間
2. **読者の不安を先に言語化して潰せ** → 冒頭1/3で疑問を3つ列挙し、本文で各個撃破
3. **「日記」ではなく「教科書」を書け** → 読者が持ち帰れる情報を必ず1つ以上含める
4. **デメリットにも正直に触れろ** → あえてマイナスを書くことで信頼度が跳ね上がる
5. **物語で感情を動かせ** → データで納得、ストーリーで決断

### 記事構成テンプレート

```
■ タイトル（30〜45字）
  → 具体的な数字 + 読者の悩みキーワード + 感情フック
  → ✕「風俗の給料について」
  → ○「コンビニ時給の5倍。私が風俗を選んだ理由と、リアルな月収」
  → ○「35歳、離婚、貯金ゼロ。子どもを守るために私が選んだ仕事」

■ リード文（200〜400字）
  → 読者の悩み・不安を3つ言語化
  → この記事で何がわかるかを明示
  → 著者の立場を明示（スタッフM・28歳・在籍2年 等）

■ 本文（h2×5〜8セクション）
  → 各セクションに「具体例 or 数字 or 体験談」最低1つ
  → セクション間にビジュアル要素（画像・装飾ボックス・テーブル）
  → 文中に自然な内部リンク（固定ページへ）
  → 3スクロールに1回は視覚的な区切り

  → つらいこと・デメリットにもあえて触れる
  → 「向いている人/向いていない人」を明示
```

### 画像ルール

- 記事あたり最低3枚（ヘッダー1 + 本文中2）
- Pillowで生成するインフォグラフィック（収入比較表・タイムライン・フロー図）
- 装飾ボックス（色付き背景+太字）でセクション区切り
- alt属性に自然な説明文

---

### 禁止事項

- 「いかがでしたか？」「まとめると」等のSEOテンプレ表現
- 噛み砕きすぎた幼稚な説明
- 根拠なしの「業界No.1」「最高水準」連呼
- 日付に依存する表現（「2026年現在」→ 使わない）
- 競合店の誹謗中傷
- 「こんにちは！店長の○○です」から始まる日記型イントロ

---

### 文体ルール

- 一人称: 「私」（語り手の人物が明確）
- 語尾: 「です・ます」+ たまに「〜だよ」「〜かな」で親近感
- 読者呼称: 「あなた」
- トーン: 友達の先輩に相談してるような安心感
- 漢字率: 30%以下（読みやすさ重視）
- 1文: 60字以内（スマホで読みやすく）
