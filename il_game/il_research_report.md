## 2. 写メ日記・PV・レビューと売上の関係

### 写メ日記の効果データ
- **7本以上/日投稿でPV 1.5倍**の実例報告あり
- ただし量より質: 10件/日で鳴らず、1件/日に減らして完売した逆事例も
- **推奨投稿頻度: 1日2〜3回**が効率的
- 約30万人商圏で約3,000人のキャストが在籍、1日約2,000本の写メ日記が投稿される競争環境
- **投稿時間帯の最適化が効果的**（ゴールデンタイムが存在）

### 正のループ構造
```
写メ日記更新 → PV増加 → ランキング上昇 → さらにPV増加 → 指名増加 → 売上増加
```

### 口コミ/レビューの影響
- 口コミゼロ・低評価の店は利用を避けるユーザーが多数
- 高評価口コミの蓄積がランキング上位と集客に直結

## 3. M&A事例と評価基準

### 価格帯
- デリヘル店の売買価格: **数百万円〜3,000万円以上**
- 具体例: 800万〜900万円での案件実績あり
- 好条件（安定売上・豊富なキャスト・会員数）は**3,000万円以上**

### 評価基準（デリヘル特有）
- 月間売上の安定性
- キャスト数・スタッフ数
- 会員数・リピーター数
- ブランド認知度・媒体ランキング
- **営業ノウハウ・運営マニュアルの有無**

### 算定方式
- 標準: **時価純資産 + 営業利益 × 2〜5年分**
- デリヘルは無店舗型で固定資産が少ない → **営業権（のれん）の比率が高い**
- 「人材（キャスト）」と「顧客基盤（会員数）」が特に重視

### M&A仲介業者
- やわらぎ倶楽部、風俗店舗M&A（fu-ma.net）、アドピン、アドガンボ
