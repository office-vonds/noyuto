## 2. 伸びているパターンTOP3

### パターン1: 速報キュレーター型（チャエン、木内）
- 海外AIニュースを毎日日本語要約 + 自分の見解
- 情報の速さ + 図解・スクショの視覚訴求
- 導線: 無料発信 → フォロワー獲得 → 法人研修・コミュニティ

### パターン2: 実践ノウハウ公開型（安達、KEITO）
- 「AIを使ってやってみた結果」を具体数値付きで公開
- 読者が真似できる粒度で書く
- 導線: 実績公開 → 信頼獲得 → コンサル契約

### パターン3: 権威ポジション確立型（深津、今井）
- 独自フレームワークを提唱（例: 深津式プロンプト）
- 新しい概念に名前をつける
- 導線: 権威 → メディア出演・書籍化 → CXO/顧問ポジション

## 4. NOYUTOの差別化ポイント

### 競合にない強み

| 要素 | 競合 | NOYUTO |
|---|---|---|
| 経営歴 | 元エンジニア・研究者が多い | **22年間のリアル経営** |
| AI活用視点 | 技術者視点 | **経営者視点** |
| 業界 | IT/Web系が大半 | **教育×AI（社会的意義）** |
| ミッション | 「効率化」「AI先進国に」 | **「戦争の種を消す教育」** |
| 組織構造 | 通常 | **Claude Code CEOという前例なし** |

### 空いているポジション

**「22年経営のおっさんが、AIにCEOを任せてみた」**

技術者がAIを語る発信は飽和。非エンジニアの現場経営者が泥臭くAIを導入する過程を見せるアカウントは極めて少ない。
