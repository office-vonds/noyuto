## STEP 1: Google Ads Editor でキャンペーン一括インポート（15分）

### 1-1. Google Ads Editor のダウンロード（初回のみ）

1. ブラウザで `ads.google.com/intl/ja/home/tools/ads-editor/` を開く
2. 「ダウンロード」→ Windows版をインストール
3. 起動 → アカウント `681-110-5790` でログイン
4. 「最新の変更を取得」をクリック（既存データ同期）

### 1-2. キャンペーンCSVインポート

1. 上部メニュー「アカウント」→「インポート」→「ファイルからインポート」
2. デスクトップの `majistretch_campaign_full_fixed.csv` を選択
3. インポートプレビュー画面が表示される
4. **確認ポイント:**
   - キャンペーン「本気ストレッチ」が表示される
   - 広告グループ6つが表示される
   - キーワード44本が表示される
   - 広告（レスポンシブ検索広告）6本が表示される
5. エラーがなければ「変更を処理」をクリック
6. 左パネルで「本気ストレッチ」キャンペーンが出現すればOK

### 1-3. 除外キーワードインポート

1. 再度「アカウント」→「インポート」→「ファイルからインポート」
2. デスクトップの `majistretch_negative.csv` を選択
3. 除外キーワード21本がプレビューされる
4. 「変更を処理」をクリック

### 1-4. サーバーに送信

1. 上部の「送信」ボタンをクリック（青いボタン）
2. 「変更内容を確認」→ 問題なければ「送信」
3. **この時点ではまだ配信されない**（キャンペーンを一時停止状態にしておくため）

**⚠️ 一時停止の確認:** 左パネルで「本気ストレッチ」キャンペーンを右クリック → 「一時停止」になっていることを確認。なっていなければ一時停止にする。

## STEP 2: 配信地域の設定（5分）

**※ これはGoogle Ads Editor ではなく、ブラウザの Google広告 管理画面で行う**

1. ブラウザで `ads.google.com` を開く
2. アカウント `681-110-5790` を選択
3. 左メニュー「キャンペーン」→「本気ストレッチ」をクリック
4. 上部「設定」タブをクリック
5. 「地域」セクションを展開
6. 「別の地域を入力する」→ 「詳細検索」をクリック
7. 「範囲」タブを選択
8. 検索欄に「山梨県甲府市上阿原町」と入力
9. 表示されたら選択 → **半径を「10 km」**に設定

**除外地域は不要**（半径10kmで自動的に範囲外は除外される）

## STEP 3: コンバージョン設定（10分）

フォーム送信完了（thanks.html表示）をコンバージョンとして計測する。

### 3-1. Google広告でコンバージョンアクションを作成

1. Google広告管理画面 → 上部「ツールと設定」（スパナアイコン）
2. 「測定」→「コンバージョン」をクリック
3. 「＋新しいコンバージョンアクション」をクリック
4. 「ウェブサイト」を選択
5. 以下を入力:

| 項目 | 設定値 |
|------|--------|
| コンバージョン名 | `予約フォーム送信` |
| カテゴリ | `予約/アポイントメント` |
| 値 | 「各コンバージョンに同じ値を割り当て」→ `10000`円 |
| カウント方法 | `初回のみ` |
| クリックスルーコンバージョンの計測期間 | `30日` |
| ビュースルーコンバージョンの計測期間 | `1日` |

6. 「作成して続行」をクリック

### 3-2. GTMでコンバージョンタグを設定

1. ブラウザで `tagmanager.google.com` を開く
2. コンテナ `GTM-K32XLKXH` を選択
3. **タグを作成:**
   - 左メニュー「タグ」→「新規」
   - タグ名: `Google広告 - 予約CV`
   - タグの種類: 「Google 広告のコンバージョン トラッキング」
   - コンバージョンID: （STEP 3-1で作成後に表示される `AW-XXXXXXXXX` の数字部分）
   - コンバージョンラベル: （同じく表示される値をコピペ）
4. **トリガーを設定:**
   - 「トリガー」→「新規」
   - トリガー名: `サンクスページ表示`
   - トリガータイプ: 「ページビュー」
   - 条件: 「一部のページビュー」
   - 条件: `Page URL` `含む` `thanks.html`
5. タグにこのトリガーを紐付けて「保存」
6. 右上「送信」→「公開」

### 3-3. 動作確認

1. GTMの「プレビュー」モードを開く
2. `https://majistretch.com/` を入力
3. フォームにテスト入力して送信
4. thanks.html に遷移した時に `Google広告 - 予約CV` タグが「発火」していればOK

## トラブルシューティング

### CSVインポートでエラーが出る場合
- 文字コードがUTF-8であることを確認（Excelで開いて保存し直すとShift-JISになることがある）
- Google Ads Editor を最新版にアップデート

### コンバージョンが計測されない場合
- GTMの「プレビュー」でthanks.htmlのタグ発火を確認
- コンバージョンIDとラベルの値が正しくコピーされているか確認
- GTMで「公開」を忘れていないか確認

### 配信されない場合
- キャンペーンのステータスが「有効」か確認
- 支払い設定が「有効」か確認（設定 → お支払い）
- 広告の審査状況を確認（通常24時間以内）
