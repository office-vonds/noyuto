# Google Ads Editor インポート用CSV

## 使い方
1. Google Ads Editor を開く
2. 「アカウント」→「インポート」→「ファイルからインポート」
3. 各CSVを読み込む
4. キャンペーン名・広告グループ名は既存のものに合わせて変更すること
5. 変更を確認 → 「送信」

## ファイル一覧
- `majistretch_keywords.csv` — 本気ストレッチ キーワード
- `stretchzero_keywords.csv` — ストレッチゼロ キーワード
- `majistretch_negative.csv` — 本気ストレッチ 除外キーワード
- `stretchzero_negative.csv` — ストレッチゼロ 除外キーワード
