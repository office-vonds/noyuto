## 口述受け入れ手順（Claude Code向け）

### NOYUTOの口述を受け取ったら:
1. 口述内容をテーマ別に分類・構造化
2. 章立てを作成（目次 → 各章の見出し → 本文）
3. 具体的な数字・実績データ・システム名を強調して配置
4. PDF/HTMLのマニュアル形式に整形
5. NOYUTOに最終確認を依頼

### 口述テーマの想定（求人AI戦略マニュアル用）:
- バニラ・ガールズヘブン自動投稿の仕組みと設定方法
- 2件の問い合わせ実績の詳細（何をしたら来たか）
- 地域別の求人SEO戦略（地域名+風俗+求人の検索構造）
- 従来の求人方法との比較（コスト・効果・工数）
- AIを使った求人文の最適化

### 口述テーマの想定（営業戦略マニュアル用）:
- ILゲームの設計思想（なぜILを上げると売上が上がるか）
- 300PV方程式の実戦的な使い方
- デリヘルタウン写メ日記戦略の具体手順
