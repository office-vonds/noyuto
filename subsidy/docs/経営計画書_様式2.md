## 2. 顧客ニーズと市場の動向

### （1）顧客ニーズ

当社の主要顧客は山梨県内を中心とした中小企業・個人事業主である。近年、以下のニーズが顕著に高まっている。

- **「SEO対策をしたいが、何から手をつければいいかわからない」** という中小企業が増加。Googleのアルゴリズム変更が頻繁に行われ、自社対応が困難になっている。
- **「WEB広告費を抑えながら、安定した集客を実現したい」** というコスト意識の高い経営者が増加。広告依存からSEO（自然検索）への移行ニーズが拡大。
- **「AIを活用したマーケティングに興味があるが、導入方法がわからない」** というAI活用への関心と、具体的な実行力不足のギャップが存在。
- **データに基づく改善提案**を求める声が増加。「感覚」ではなく「数値」で成果を示すコンサルティングが求められている。

### （2）市場の動向

- 国内SEO市場は年率10%以上で成長を続けており、特に地方中小企業のDX推進に伴いSEOコンサルティングの需要は拡大傾向にある。
- 山梨県内においては、全日本SEO協会認定コンサルタントは当社代表のみであり、専門性の高いSEOサービスを提供できる事業者が極めて少ない。
- 一方で、東京圏の大手SEO会社が地方市場にも参入しており、価格競争・サービス品質の両面で競争が激化している。
- 生成AI（ChatGPT等）の普及により、「AIを活用したSEO分析・コンテンツ制作」への需要が急増。従来型のSEOコンサルから、AI活用型コンサルへの転換が業界全体のトレンドとなっている。

## 3. 自社や自社の提供する商品・サービスの強み

### 強み

1. **山梨県唯一のSEO協会認定コンサルタント** — 全日本SEO協会の認定資格を保有し、最新のGoogleアルゴリズム変動情報をいち早く入手・対応できる。
2. **上位表示達成率90%以上の実績** — 「お悔み」「ビニール加工」「モデル 募集」等、競合の多いキーワードで1位を獲得した実績多数。
3. **20年以上の業界経験** — 2004年から一貫してWEBマーケティングに従事しており、SEOの本質を理解した戦略設計が可能。
4. **ワンストップ提供** — SEO戦略立案からサイト制作・コンテンツ制作・写真撮影まで一社で完結。クライアントの発注コスト・管理コストを削減。
5. **AI活用の先行実践** — Claude Code等の生成AIを実業務に導入済みで、AI×SEOの知見を蓄積中。

### 課題

1. **代表の属人性が高い** — SEOコンサルティングの品質が代表・小沢個人のスキルに依存しており、代表が稼働しないと売上が立たない構造になっている。
2. **新規顧客開拓の仕組み不足** — 既存顧客からの紹介・口コミに依存しており、能動的な新規開拓チャネルが弱い。
3. **レポーティング業務の負荷** — クライアントへのSEOレポート作成に手作業が多く、月間20〜30時間を費やしている。この時間を営業・コンサルに充てたい。
