## 1. 補助事業で行う事業名

**「AI活用型SEO分析自動化システム導入とホームページリニューアルによる新規顧客開拓事業」**

---

## 2. 販路開拓等（生産性向上）の取組内容

### （1）事業の概要

当社は山梨県甲府市を拠点にSEOコンサルティングを主軸としたWEBマーケティング事業を展開しているが、代表の属人性が高く、新規顧客開拓の仕組みが不十分という経営課題を抱えている。

本補助事業では、以下の2つの取り組みにより、**新規顧客の獲得チャネルを構築し、持続的な売上拡大を実現する。**

### （2）取組内容

#### 取組① 自社ホームページのリニューアル（販路開拓）

**【現状の課題】**
現在の自社ホームページは情報量が少なく、SEOコンサルティングの具体的な成果事例・サービス内容が十分に伝わっていない。そのため、検索経由での新規問い合わせが月2〜3件に留まっている。

**【実施内容】**
- サービスページの全面刷新（SEOコンサル・WEB制作・AI活用サービスの3本柱を明確化）
- **成功事例・実績ページの新設**（具体的なキーワード順位改善データ、クライアントの声を掲載）
- **問い合わせ導線の最適化**（全ページにCTAを設置、無料SEO診断フォームの設置）
- **SEOブログの新設**（AI活用によるコンテンツ定期発信で検索流入を増加）
- スマートフォン対応の完全最適化（モバイルファースト設計）
- ページ表示速度の改善（Core Web Vitals対応）

**【期待効果】**
- 検索経由の新規問い合わせ: 月2〜3件 → 月8件（約3倍）
- ホームページ経由の新規成約: 月0〜1件 → 月2〜3件

#### 取組② AI活用型SEO分析・レポート自動化システムの導入（業務効率化＋販路開拓）

**【現状の課題】**
クライアントへのSEO分析レポート作成に月間約30時間を費やしており、この業務負荷が新規営業活動の時間を圧迫している。また、レポートの品質が代表のスキルに依存しており、属人的な業務となっている。

**【実施内容】**
- **AI搭載SEO分析ツールの導入**（検索順位変動・競合分析・キーワード提案の自動化）
- **自動レポート生成システムの構築**（クライアントごとの月次SEOレポートを自動作成）
- **AIチャットボットによる初期SEO診断機能**（ホームページ訪問者が自社サイトのSEO課題を無料で診断可能に）

**【期待効果】**
- レポート作成時間: 月30時間 → 月5時間（約83%削減）
- 削減された25時間を新規営業・提案活動に充当
- 無料SEO診断から見込み客リストを獲得（月10〜20件見込み）

#### 取組③ 販促物の制作・広告出稿

**【実施内容】**
- サービス案内チラシ・パンフレットの制作（商工会・金融機関への設置用）
- Google広告の出稿（「山梨 SEO対策」「甲府 ホームページ制作」等の地域キーワード）
- セミナー告知用のLP（ランディングページ）制作

**【期待効果】**
- 商工会経由の紹介: 月1〜2件
- 広告経由の問い合わせ: 月3〜5件

## 4. 補助事業の効果

### （1）売上への効果

| 指標 | 現状（補助事業前） | 目標（補助事業後1年） | 増加率 |
|------|--------------------|-----------------------|--------|
| 月商 | 約98万円 | 130万円 | +33% |
| 年商 | 約1,178万円 | 1,560万円 | +33% |
| 新規問い合わせ数 | 月2〜3件 | 月15件 | +500% |
| 新規成約数 | 月0〜1件 | 月3〜4件 | +300% |
| 顧客数 | 約15社 | 20社 | +33% |

### （2）定性的効果

- **属人性の解消**: AIツール導入により、代表以外でもSEO分析・レポート作成が可能となり、事業の持続可能性が向上。
- **サービス品質の向上**: AI分析による客観的データに基づくコンサルティングで、クライアント満足度が向上。
- **新サービスの創出**: 「AI×SEO診断パッケージ」として新たな商品化が可能。従来より低価格で提供でき、中小企業の裾野を拡大。
- **地域経済への貢献**: 山梨県内中小企業のDX推進・WEB集客力向上を支援し、地域経済の活性化に貢献。
