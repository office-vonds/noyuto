## 1. 設立に必要な書類一覧

### 法務局に提出する書類

| # | 書類名 | 備考 |
|---|--------|------|
| 1 | 合同会社設立登記申請書 | 法務局の書式に従う |
| 2 | 定款 | 公証人の認証は不要（株式会社と異なる） |
| 3 | 代表社員の印鑑届書 | 法人実印の届出 |
| 4 | 代表社員の印鑑証明書 | 発行3か月以内のもの |
| 5 | 払込証明書 | 出資金の払込を証明する書面（通帳コピー添付） |

## 2. 設立費用の概算

### 法定費用

| 項目 | 紙の定款 | 電子定款 |
|------|---------|---------|
| 登録免許税 | 6万円（資本金857万円以下の場合） | 6万円 |
| 定款の収入印紙代 | 4万円 | **0円** |
| 定款認証手数料 | 0円（合同会社は認証不要） | 0円 |
| **法定費用合計** | **10万円** | **6万円** |

※登録免許税の計算：資本金 × 0.7%（6万円に満たない場合は6万円）
- 資本金100万円 → 6万円
- 資本金1,000万円 → 7万円
- 資本金2,000万円 → 14万円

| 司法書士への依頼（任意） | 5〜10万円 |
| 税理士への届出代行（任意） | 2〜5万円 |

### 設立費用の総額目安

| パターン | 費用目安 |
|----------|---------|
| 全て自分でやる（電子定款） | 約7〜8万円 |

| 司法書士に依頼（電子定款） | 約12〜18万円 |

## 3. 設立の手順（時系列）

### STEP 1: 事前準備（設立1〜2週間前）

- [ ] 会社の基本事項を決定する
  - 商号（会社名）：「合同会社○○」
  - 本店所在地
  - 事業目的
  - 資本金額
  - 社員構成（出資者）
  - 事業年度（決算月）
- [ ] 商号調査を行う（法務局 or 国税庁法人番号公表サイトで同一商号の確認）
- [ ] 法人実印を作成する（商号確定後すぐに発注）
- [ ] 代表社員の印鑑証明書を取得する（市区町村窓口 or コンビニ交付）

### STEP 2: 定款の作成（設立1週間前）

- [ ] 定款を作成する（後述「4. 定款に記載すべき事項」参照）
- [ ] 電子定款にする場合：電子署名を付与する（or 司法書士に依頼）
- [ ] 紙の定款にする場合：収入印紙4万円を貼付する
- [ ] 社員全員が定款に記名押印する（紙の場合）

### STEP 3: 出資金の払込（定款作成後）

- [ ] 代表社員個人の銀行口座に出資金を振り込む
  - 振込人名義は各社員の名前で行う
  - 定款作成日以降に振込を行う
- [ ] 通帳の表紙・見開き・振込記載ページをコピーする
- [ ] 払込証明書を作成し、通帳コピーと合綴する

### STEP 4: 登記申請（設立日 = 申請日）

- [ ] 設立登記申請書を作成する
- [ ] 印鑑届書を記入する
- [ ] 必要書類を全て揃え、管轄法務局に提出する
  - 窓口持参 or 郵送 or オンライン申請（登記・供託オンライン申請システム）
  - **申請日 = 会社設立日**になるため、設立日を選んで申請する
- [ ] 登録免許税を納付する（収入印紙 or 電子納付）
- [ ] 登記完了を確認する（通常1〜2週間）

### STEP 5: 登記完了後の手続き（設立後すぐ）

- [ ] 登記事項証明書を取得する（3〜5通）
- [ ] 印鑑カードを取得する
- [ ] 印鑑証明書（法人）を取得する
- [ ] 各種届出を行う（後述「7. 届出一覧」参照）
- [ ] 法人銀行口座を開設する

## 4. 定款に記載すべき事項

### 絶対的記載事項（必須）

これらの記載が欠けると定款自体が無効になる。

| # | 項目 | 記載例 |
|---|------|--------|
| 1 | 目的 | AI技術を活用した教育サービスの企画・開発・運営、ソフトウェアの設計・開発・販売、等 |
| 2 | 商号 | 合同会社○○ |
| 3 | 本店の所在地 | 東京都○○区（最小行政区画まででOK） |
| 4 | 社員の氏名または名称及び住所 | 社員 小沢宗弘 東京都○○区... |
| 5 | 社員全員が有限責任社員である旨 | 「社員の全部を有限責任社員とする」 |
| 6 | 社員の出資の目的及びその価額 | 金銭出資 金○○万円 |

### 相対的記載事項（定款に記載しないと効力が生じない）

| # | 項目 | 説明 |
|---|------|------|
| 1 | 業務執行社員の定め | 複数社員がいる場合、誰が業務執行するか |
| 2 | 代表社員の定め | 複数社員がいる場合、代表者を指定 |
| 3 | 利益の配分割合 | 出資比率と異なる配分にする場合 |
| 4 | 社員の加入・退社の条件 | デフォルトは全員同意 |
| 5 | 残余財産の分配方法 | 解散時の財産分配ルール |
| 6 | 競業避止義務の免除 | 社員が他の事業を行う場合 |

### 任意的記載事項（記載推奨）

| # | 項目 | 推奨内容 |
|---|------|---------|
| 1 | 事業年度 | 「毎年○月1日から翌年○月末日まで」 |
| 2 | 社員の報酬 | 「社員総会の決議により定める」等 |
| 3 | 定款変更の手続 | 社員の過半数 or 全員の同意 |
| 4 | 解散事由 | 任意に定めることが可能 |

### 事業目的の記載例（AI事業向け）

```
1. AI技術を活用した教育サービスの企画、開発及び運営
2. コンピュータソフトウェアの設計、開発、販売及び保守
3. インターネットを利用した情報提供サービス
4. ゲームコンテンツの企画、開発及び販売
5. コンサルティング業務
6. 前各号に附帯又は関連する一切の事業
```

※将来の事業展開を見据えて広めに記載しておくこと。追加時は定款変更が必要になる。

## 5. 資本金の決め方

### 法的な最低額

- 合同会社の最低資本金は**1円**（ただし実務上は非推奨）

### 資本金額の決定基準

| 検討項目 | 内容 |
|----------|------|
| 初期運転資金 | 3〜6か月分の運転資金を目安にするのが一般的 |
| 消費税の免税 | 資本金**1,000万円未満**で設立すれば最大2年間消費税免税（※条件あり） |
| 法人住民税均等割 | 資本金1,000万円以下：年間7万円、1,000万円超：年間18万円〜 |
| 許認可の要件 | 事業によっては資本金の最低額が定められている場合あり |
| 銀行口座開設 | 資本金が少なすぎると（1〜10万円等）口座開設の審査で不利 |
| 取引先の信用 | BtoB取引では資本金が少ないと信用面で不利な場合がある |

### 推奨額

| パターン | 資本金額 | 理由 |
|----------|---------|------|
| 最小限で始める | 50〜100万円 | 口座開設に困らない最低ライン |
| 標準的 | 100〜300万円 | 信用面と税務のバランスが良い |
| 余裕をもたせる | 300〜999万円 | 消費税免税を活かしつつ信用確保 |

### 注意事項
- 資本金1,000万円以上で設立すると**設立初年度から消費税課税事業者**になる
- 資本金は登記事項であり、増減には法的手続きが必要
- 出資金の払込は設立前に代表社員の個人口座に振り込む

## 6. 決算月の選び方

### 決算月を選ぶ基準

| 基準 | 説明 |
|------|------|
| **繁忙期を避ける** | 事業の繁忙期と決算業務が重ならないようにする |
| **設立月の前月** | 設立初年度の事業年度を最長（約12か月）にできる |
| **資金繰り** | 法人税・消費税の納付時期と資金需要が重ならないようにする |
| **消費税免税期間の最大化** | 設立月の前月を決算月にすると免税期間が最長になる |
| **3月・12月を避ける** | 税理士の繁忙期と重なり、対応が手薄になる可能性 |

### 具体例

4月に設立する場合：
- 決算月を**3月**にすると → 初年度は約12か月（最大）
- 決算月を**9月**にすると → 初年度は約6か月（短い）

### 法人税等の納付期限

| 税目 | 納付期限 |
|------|---------|
| 法人税 | 事業年度終了日の翌日から**2か月以内** |
| 法人住民税 | 同上 |
| 法人事業税 | 同上 |
| 消費税 | 事業年度終了日の翌日から**2か月以内** |

※例：3月決算 → 5月末が申告・納付期限

| 6 | 源泉所得税の納期の特例の承認に関する申請書 | 税務署 | 随時（給与支給人員10人未満の場合） |

| 3 | 被扶養者（異動）届 | 年金事務所 | 事実発生日から**5日以内** |

## 8. 銀行口座開設の注意点

### 口座開設に必要な書類

| # | 書類 |
|---|------|
| 1 | 登記事項証明書（発行6か月以内） |
| 2 | 定款のコピー |
| 3 | 法人の印鑑証明書 |
| 4 | 代表社員の本人確認書類（運転免許証 or マイナンバーカード等） |
| 5 | 法人番号指定通知書 or 法人番号が確認できる書類 |
| 6 | 法人の実印・届出印 |
| 7 | 事業内容がわかる資料（会社案内、ホームページ等） |

### 合同会社で口座開設する際の注意点

| 注意点 | 対策 |
|--------|------|
| 合同会社は株式会社より審査が厳しい傾向 | 事業計画書を準備し、事業内容を明確に説明できるようにする |
| 資本金が少ないと審査に不利 | 最低でも50万円以上の資本金が望ましい |
| 設立直後は審査が通りにくい場合がある | ネット銀行（GMOあおぞら、住信SBIネット等）は比較的開設しやすい |
| バーチャルオフィスだと断られることがある | 実態のあるオフィスの方が有利 |
| ホームページがないと不利 | 簡易的でもいいので事業内容を記載したサイトを用意する |

### 銀行選びのポイント

| 銀行の種類 | メリット | デメリット |
|-----------|---------|------------|
| メガバンク | 社会的信用が高い、融資に有利 | 審査が厳しい、手数料が高い |
| 地方銀行・信用金庫 | 地域密着で相談しやすい | 店舗が限られる |
| ネット銀行 | 審査が比較的通りやすい、手数料が安い、24時間利用可 | 窓口がない、融資に不利な場合がある |

### 推奨：複数口座の開設

1. **メイン口座**：ネット銀行（日常の入出金・振込用）
2. **サブ口座**：メガバンク or 地銀（信用面・融資対策用）

## 9. 設立後すぐにやるべきこと

### 設立直後（1週間以内）

- [ ] 登記事項証明書を取得（3〜5通）
- [ ] 法人の印鑑証明書を取得（2〜3通）
- [ ] 印鑑カードを取得
- [ ] 法人番号を確認（国税庁法人番号公表サイトで自動付番）
- [ ] **社会保険の届出**（健康保険・厚生年金：5日以内が期限）

### 設立後2週間以内

- [ ] 法人銀行口座の開設手続き
- [ ] 税務署への届出（法人設立届出書）
- [ ] 都道府県税事務所・市区町村への届出（15日以内）
- [ ] 青色申告の承認申請書の提出（3か月以内だが早めに）
- [ ] 給与支払事務所等の開設届出書の提出

### 設立後1か月以内

- [ ] 会計ソフトの導入と初期設定（freee, マネーフォワード, 弥生 等）
- [ ] 経理体制の整備（勘定科目の設定、経費精算ルール等）
- [ ] 顧問税理士の選定・契約
- [ ] インボイス登録の検討（課税事業者になる場合）
- [ ] 名刺・ホームページの作成
- [ ] ドメイン取得・メールアドレスの設定

### 設立後3か月以内

- [ ] 源泉所得税の納期の特例の申請（10人未満の場合、年2回の納付にできる）
- [ ] 就業規則の作成（従業員10人以上の場合は義務）
- [ ] 各種保険の検討（賠償責任保険、火災保険等）
- [ ] 中小企業共済・小規模企業共済の加入検討（節税対策）

### 継続的にやること

- [ ] 毎月の記帳・経理処理
- [ ] 源泉所得税の納付（毎月10日 or 半年に1回）
- [ ] 社会保険料の納付（毎月末日）
- [ ] 年末調整（12月）
- [ ] 法定調書の提出（1月末）
- [ ] 決算・確定申告（事業年度終了から2か月以内）

## 設立スケジュール例（4月設立の場合）

| 時期 | やること |
|------|---------|
| 3月第3週 | 基本事項決定、商号調査、印鑑発注 |
| 3月第4週 | 定款作成、印鑑証明書取得 |
| 4月第1週 | 出資金払込、登記書類準備 |
| 4月1日（設立日） | 法務局に設立登記申請 |
| 4月第2週 | 登記完了確認、謄本・印鑑証明書取得 |
| 4月第2週 | 社会保険届出（5日以内）、銀行口座開設 |
| 4月中 | 税務署・都道府県への届出 |
| 5月中 | 会計ソフト導入、税理士契約 |
