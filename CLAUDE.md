## セッションルール
- タスク完了後、関連する次のタスクがあれば確認なしで自動着手
- エラー発生時は3回リトライしてから報告
- ファイル作成・編集後は自動でgit add && git commit
- 「〇〇を作って」の指示にはSKILL.mdを自動参照してから着手
- 中間報告は不要。最終結果のみ報告
- 不明点は推測で最適解を選び、判断根拠を末尾に1行添える
