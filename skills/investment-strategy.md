> **最終判断者:** NOYUTO
> **証券口座:** SBI証券 / 野村証券
> **基本思想:** リスクなきものに利益なし

---

## 投資哲学

- 市場が確信する前に動く。全員が知ってからでは遅い
- データと根拠で判断する。感覚・噂・煽りでは動かない
- 小さく入り、検証し、確信が上がれば追加する
- 間違いは早く認める。損切りは恥ではなく技術

---

## 確信度別アクションルール

| 確信度 | アクション | ポジションサイズ |
|--------|-----------|----------------|
| 70〜80% | エントリー（小さく入る） | 総資産の5〜10% |
| 80〜90% | 追加投入 | 最大20%まで |
| 90%以上 | フルポジション検討 | 最大30%まで |
| 70%未満 | 見送り。監視リストに入れる | 0% |

### 確信度70%の定義

以下のうち**4つ以上**を満たすこと：

1. マクロ環境が追い風（金利・政策・景気サイクル）
2. ファンダメンタルが健全（財務・成長性・バリュエーション）
3. テクニカルがエントリーポイントを示唆（トレンド・出来高）
4. カタリスト（材料）が明確に存在する
5. リスクリワード比が2:1以上（利益幅がリスクの2倍以上）
6. 自分で説明できる（なぜ買うか3行で言えるか）

## リスク管理ルール

### 損切り
- 個別ポジション: **-10%で損切り**（例外なし）
- ポートフォリオ全体: **月間-15%で全ポジション見直し**

### ポジション上限
- 単一銘柄/資産: 総資産の**最大30%**
- 単一セクター: 総資産の**最大40%**
- 暗号資産全体: 総資産の**最大20%**（ボラティリティ考慮）

### 禁止事項
- 感情売買（怒り・焦り・FOMO）
- SNSの煽りだけを根拠にしたエントリー
- 損切りラインの後出し変更
- 確信度70%未満での見切り発車
- 借金してのレバレッジ投資

## 投資判断フロー

```
清水悦夫（アナリスト）
  │ 日次: データ収集・分析・レポート提出
  ▼
小丸（リーダー）
  │ 確信度スコアリング（6項目チェック）
  │ 戦略提案書作成（選択肢＋推奨案＋根拠）
  ▼
KIRYU（COO）
  │ 事業全体との整合性チェック
  │ 投入可能資金の確認
  ▼
NOYUTO（CEO）
  │ 最終判断: GO / NO GO / 保留
  ▼
実行 → モニタリング → 定期レビュー
```

## 定期レビュー

| 頻度 | 内容 |
|------|------|
| 日次 | 清水の市場サマリー確認 |
| 週次 | ポートフォリオ損益チェック・小丸の戦略レビュー |
| 月次 | 資産配分の見直し・確信度の再評価 |
| 四半期 | 投資方針そのものの振り返り・ルール改定検討 |
