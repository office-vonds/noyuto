## 事業概要

山梨県甲府市を拠点とするWEBマーケティング会社。2004年からネットビジネスに従事し、SEO対策を中心に事業を展開。5年連続で売上高1億円以上を達成した実績あり。

### 事業内容
- SEOコンサルティング（メイン事業）
- アクセスアップコンサルティング
- WEBサイト制作（SEO込みの集客サイト）
- 写真撮影
- グラフィック作成（広告物全般）
- アプリ＆システム開発
- 電気工事業

### 強み
- 一般社団法人全日本SEO協会認定SEOコンサルタント（山梨県唯一）
- 上位表示達成率90%以上
- 「集客に結び付ける」までをワンストップ提供
- Googleアルゴリズム変動への迅速対応（全日本SEO協会所属）
- 第一種電気工事士資格保有

## 現在のビジネスモデルと今後の方針

**現在:** WEBコンサル・開発・制作を行い、クライアント企業を稼がせるビジネス展開。

**課題:** 小沢宗弘の属人性が高い。社長が動かないと売上が立たない構造。

**今後の方針:** 小沢宗弘の属人性を排除したビジネス展開への転換。
- 仕組み化・システム化による脱属人
- 再現可能なサービスモデルの構築
- Claude Code / AIを活用した業務自動化
